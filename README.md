# Praxis Aktivraum Website

Mehrseitige, statische Website für die Physiotherapie-Praxis "Praxis Aktivraum".

## Starten

```bash
npm install
npm run dev
```

## Seiten

- Start
- Leistungen
- Über uns
- Team
- Kontakt

Hinweis: Bildplatzhalter sollten später durch echte Praxisfotos ersetzt werden.
